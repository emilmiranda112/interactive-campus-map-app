# Interactive Campus Map 📍

An interactive, mobile-responsive web application designed to help students, staff, and visitors seamlessly navigate Rancho Buena Vista Highschool. Built entirely in Python using the Anvil framework and integrated with the Google Maps API.

## 🚀 Features
* **Dynamic Marker Plotting (Planned):** Instantly view key campus buildings and landmarks.
* **Interactive Info Windows (Planned):** Click any marker to view descriptions, photos, or quick facts about the location.
* **Smart Search(Planned):** Search bar functionality that auto-pans and zooms to selected campus destinations.
* **Mobile-Responsive(Planned):** Optimized UI/UX designed to work flawlessly on both desktop monitors and smartphones.

## 🛠️ Tech Stack
* **Frontend & Backend:** [Anvil.works]([Live app link - pending deployment]) (Full-stack Python Web Framework)
* **API:** Google Maps JavaScript API
* **Version Control:** GitHub
